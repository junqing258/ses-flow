# SES Flow 帮助中心

欢迎使用 Workflow Builder。这里会逐步补充编辑器说明、节点能力、运行调试和发布流程等帮助内容。

## 快速开始

1. 回到[工作流首页](/#/workflow-list)创建一个新的 workflow。
2. 在编辑器里添加节点、连接流程，并保存草稿。
3. 通过运行记录检查执行结果，再按需发布版本。
